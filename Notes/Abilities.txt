*****************************
SET THEMING FRAMEWORK
***************************
But in any case I would like the items to be grouped thematically, like maybe a demonic themed set and a nature themed set, etc. Items from a set should generally have more synergy with other items from their set than with any other set, but there should be enough synergy with other sets that not every game feels like "pick a set and stick with it". 

*"Black sheep" items will also be expected; a few bumps in the road to complicate the meta.

*"married" sets may be interesting, or set pairs that have more synergy with their spouse set than with their own.

*I'm also toying with the idea of elemental effects and elemental weaknesses/resistances. This would be a good way to discourage over-reliance on a single set and lends itself easily to sets (electric, ice, fire, dark, etc.) I could also add a little more structure by having no sets with a single element, but every set has 2 elements, like a thunderstorm set that has water and electric.

*Megaloot/TFT style set bonuses are not out of the question. They're also a good way to discourage over-reliance on single sets

*Many if not all sets should tend toward an archetype like crit chance
