# Set Notes

## General Theme
This set (along with the dark set) was inspired by Kayle and Morgana: https://universe.leagueoflegends.com/en_GB/story/champion/kayle/
Not just the light/darkness theme, but that they are in some ways very similar and in others polar opposites, with nothing in between.
The light set is centered around holiness, light (in the literal sense), and to a degree, thermonuclear fusion.

## Colors
Primarily white and light yellow, perhaps with some light blue.
