# Set Notes

## General Theme
This set (along with the light set) was inspired by Kayle and Morgana: https://universe.leagueoflegends.com/en_GB/story/champion/morgana/
Not just the light/darkness theme, but that they are in some ways very similar and in others polar opposites, with nothing in between.
The darkness set is centered around darkness (obviously), death, void, and cold.

## Colors
Primarily black and purple, typically purple in the center and black in the extremities.

## Relevant gameplay elements
Execute (instant death at low health)
Blind
Chill (reduce movement speed)
